# Use Lifetime Purchase Instead of Platinum Try-out Tier

Paid Try-outs use a single paid content state and can be unlocked either through an active Premium Membership or a Lifetime Try-out Purchase. We no longer model Platinum as a separate Try-out tier because it describes a monetization path, not a distinct content tier; this keeps catalog filters, admin setup, and access rules focused on content access while preserving per-Try-out lifetime purchases as Products and Entitlements.
