# Finalize Weekly Leaderboards for Top-N Badges

Top-N Badge awards come from a finalized weekly leaderboard snapshot, not from the live in-week leaderboard or a daily recomputation. We chose this because Top-N Badges represent a completed weekly competition: the system needs a stable audit trail for rank, counted EXP, participant threshold, and awarded Badges, while daily Badge evaluation remains appropriate for continuously-achievable Badges such as streak, level, progress, and score Badges.
